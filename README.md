# PC Build & Troubleshooting — Khanna Consulting

Internal knowledge base for PC build procedures, troubleshooting guides, and resolution notes.

## Structure

```
builds/          — Build guides and hardware configs
troubleshooting/ — Issue diagnosis and resolution docs
parts/           — Parts lists, compatibility notes, vendor info
```

## Usage

Each markdown file documents a specific situation or procedure. Name files descriptively:

- `builds/client-smith-gaming-rig.md`
- `troubleshooting/boot-loop-after-bios-update.md`
- `parts/gpu-compatibility-matrix.md`
